# Elastic Distribution of OpenTelemetry Browser

This repository holds browser packages authored by Elastic for using
and working with OpenTelemetry. Primarily that includes
[`@elastic/opentelemetry-browser`](./packages/opentelemetry-browser), the
Elastic Distribution of OpenTelemetry Browser (or "EDOT Browser" for short).
